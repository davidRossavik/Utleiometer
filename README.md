# Dramble

Dramble is a real-time multiplayer party game built with React Native, Expo, and TypeScript.  
The game combines challenge-based gameplay with strategic betting mechanics in a structured, round-based system.

The architecture is designed for synchronized multiplayer interaction, scalable challenge handling, and modular game state management.

---

## Overview

Dramble allows multiple players to join a shared session and participate in interactive game rounds consisting of:

- A betting phase
- A challenge phase
- A resolution phase

The system ensures real-time updates across all connected players using Supabase for backend services and state synchronization.

---

## Core Features

- Real-time multiplayer synchronization
- Structured round lifecycle management
- Modular challenge handling system
- Strategic betting mechanics
- Host-controlled game flow
- Scalable and maintainable architecture

---

## Tech Stack

**Frontend**
- React Native
- Expo
- TypeScript

**Backend / Infrastructure**
- Supabase (Realtime, Database, Authentication)

**Architecture Principles**
- Component-based UI structure
- Centralized game state management
- Modular challenge logic
- Clear separation between UI and game logic
